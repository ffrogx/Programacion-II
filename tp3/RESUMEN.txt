CONCEPTOS APLICADOS
====================

✅ HERENCIA
   - Uso en: Ejercicios 1, 2, 5
   - Permite reutilización de código base
   - Jerarquías de clases bien definidas

✅ CLASES ABSTRACTAS
   - Uso en: Ejercicios 1, 2, 5
   - Definen contratos sin implementación
   - Previenen instanciación directa
   - Métodos abstractos fuerzan implementación en subclases

✅ INTERFACES
   - Uso en: Ejercicios 3, 4, 5
   - Definen comportamientos adicionales
   - Permiten herencia múltiple de comportamientos
   - Desacoplamiento entre componentes

✅ POLIMORFISMO
   - Procesamiento de colecciones heterogéneas
   - Llamadas a métodos específicos según tipo real
   - Reducción de complejidad de código cliente
